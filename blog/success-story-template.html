この記事のポイント

手数料10%の低コスト紹介で、病院も看護師もWin-Winの転職を実現した事例をご紹介します。大手エージェントの手数料30%と比較して、病院の採用ハードルが下がることで条件交渉がしやすくなり、看護師にとっても有利な結果につながります。

「手数料10%」が生み出すWin-Winの好循環

看護師の転職において、転職エージェントの手数料は「見えないコスト」として大きな影響を与えています。業界平均の20-30%に対して、ナースロビーが提供する手数料10%のモデルは、病院と看護師の双方にメリットをもたらすWin-Winの仕組みです。

この記事では、手数料10%がどのようにして「三方良し」（病院良し、看護師良し、紹介会社良し）を実現するのか、具体的なシミュレーションと成功パターンを交えて解説します。

※本記事の事例は、想定されるケースに基づくシミュレーションです。

コスト比較：手数料10%と30%の差額が生む効果

1人の採用コスト比較

項目大手A社（手数料30%）ナースロビー（手数料10%）差額
看護師の年収480万円480万円-
紹介手数料144万円48万円96万円

たった1人の採用で、105.6万円の差額が生まれます。この金額が病院にとって何を意味するか、具体的に見てみましょう。

差額105.6万円でできること

  • 既存看護師の賞与に1人あたり5万円上乗せ（20名分）
  • ナースステーションの休憩スペース改善
  • 院内研修の外部講師招聘費用（3-4回分）
  • 電子カルテのオプション機能追加
  • 看護師向け福利厚生の充実（食堂の改善、制服の更新など）

つまり、手数料を抑えることで、その差額を看護師の働く環境の改善に充てることができるのです。これこそが「Win-Win」の本質です。

成功パターン1：急性期病院 × 中堅看護師

シミュレーション事例

横浜の大病院から小田原の中規模病院へ

看護師プロフィール：28歳女性、急性期病院で経験6年、外科病棟勤務

転職前：横浜市内の大学病院勤務。年収520万円。通勤1時間、夜勤月5回。人間関係のストレスが大きく、転職を検討。

転職後：小田原市の中規模病院（150床）。年収480万円。通勤15分（車）、夜勤月4回。

ポイント：

  • 年収は40万円減だが、家賃が月3万円安くなり年間36万円の実質節約
  • 通勤時間が1時間→15分に短縮。往復で1.5時間の自由時間を獲得
  • 手数料10%（48万円）なので病院側の採用コスト負担が軽く、入職後のプレッシャーも少ない
  • 小規模ゆえに職場の風通しが良く、人間関係のストレスが大幅に軽減

病院側のメリット

この事例で病院が得たメリットは以下の通りです。

  • 大手エージェント経由なら144万円の手数料が、48万円で済んだ（96万円の節約）
  • 節約分の一部を、既存スタッフの処遇改善に充当
  • 急性期6年の経験者を獲得し、即戦力として配属
  • 手数料負担が軽いため、「紹介会社経由の看護師」に対する心理的なハードルが低い

成功パターン2：訪問看護へのキャリアチェンジ

シミュレーション事例

夜勤のある病院から訪問看護ステーションへ

看護師プロフィール：33歳女性、経験10年、内科病棟。未就学児2人の母。

転職前：東京都内の総合病院勤務。年収550万円。夜勤月4回。保育園の送迎に間に合わず、夫との家事分担もストレスに。

転職後：秦野市の訪問看護ステーション。年収460万円。日勤のみ、土日休み。オンコール月2回。

ポイント：

  • 年収は90万円減だが、夜勤手当がなくなった分がほぼ全体の差額
  • 秦野の家賃は東京より月5万円安く、年間60万円の節約
  • 実質的な生活水準は同等以上に維持
  • 保育園のお迎えに余裕ができ、家族との時間が増加
  • 訪問看護の自律性の高い働き方にやりがいを感じている

なぜ手数料10%が実現できるのか

「安かろう悪かろう」ではないかと疑問に思う方もいるでしょう。ナースロビーが手数料10%を実現できる理由を、正直にお伝えします。

1. AIの活用によるオペレーション効率化

求人情報の整理、条件マッチングの下準備、スケジュール調整など、AIで効率化できる業務はAIに任せています。これにより、少人数でも高品質なサービスを提供できます。ただし、看護師さんとの面談、病院との条件交渉など、人間にしかできない業務は人間が担当します。

2. 広告費を最小化

大手エージェントは年間数十億円の広告費を投じています。ナースロビーは、SNSやSEO、口コミなど、低コストの集客手法に特化することで、広告費を大幅に削減しています。

3. 地域特化による効率性

全国展開ではなく、神奈川県西部に特化することで、地域の病院事情に精通し、効率的なマッチングが可能です。「広く薄く」ではなく「狭く深く」サービスを提供するモデルです。

4. 適正利益の追求

ナースロビーは、過度な利益追求よりも、地域の医療に貢献するという理念を持っています。手数料10%でも事業として持続可能な経営が成立するよう、コスト構造を設計しています。

Win-Winが生む好循環

手数料10%のモデルが生み出す好循環をまとめます。

  1. 病院の採用コストが下がる → 経営にゆとりが生まれる
  2. 浮いたコストが看護師の待遇改善に充てられる → 給与アップ、教育充実、職場環境改善
  3. 看護師の満足度が上がる → 離職率が下がる
  4. 離職率が下がる → 採用コストがさらに減る（好循環）
  5. 入職した看護師が口コミで紹介 → 新たな看護師が来る
  6. 地域の医療が安定する → 地域住民にもメリット

これは理想論ではなく、低手数料モデルが持続可能な仕組みとして機能する論理的な根拠です。ナースロビーは、この好循環を神奈川県西部で実現することを目指しています。

まとめ：手数料を下げることは、医療の質を上げること

看護師転職エージェントの手数料は、単なる「紹介会社の取り分」ではありません。その金額は、病院の経営、看護師の待遇、そして最終的には患者へのケアの質にまで影響を与えます。

手数料を10%に抑えることは、この連鎖のすべてをポジティブな方向に動かす力を持っています。ナースロビーは、AIと地域密着を武器に、この「三方良し」のモデルを実現し続けます。
